## 🏡 About

Desa Digital System is a backend application designed to simulate a modern village management system. 
It focuses on organizing administrative processes such as citizen management, regional hierarchy (RT/RW), document submissions, and announcements in a structured and scalable way.

This project aims to digitize traditional village services by providing a centralized system where residents, RT, RW, and village leaders can interact efficiently through role-based access and well-defined workflows.

Built with a strong emphasis on clean architecture, real-world system design, and backend best practices, this project reflects how technology can improve public service at the village level.
